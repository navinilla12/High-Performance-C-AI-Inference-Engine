# High-Performance C++ AI Inference Engine

## Overview
This project simulates a lightweight AI inference runtime written in C++ focusing on computational efficiency and CPU-level optimization.

## Features
- Matrix multiplication engine optimized for memory locality
- ReLU activation pipeline
- Execution benchmarking
- Floating-point computation behavior analysis

## Tech Stack
C++, Linux, Python (benchmarking), NumPy

## How to Run
g++ inference_engine.cpp -O3 -o engine
./engine

## Output
Displays inference execution time in seconds.

## Goal
Understand how AI inference behaves at CPU level and how optimization impacts runtime performance.
